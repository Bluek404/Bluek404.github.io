Shields.io —— 制作各种状态图标

嘛这篇文章其实是凑数的

2016-07-27 00:00

在 github 上看见很多项目都有 这样的图标对不对？

嗯，这就是今天我在好不容易重新开始的 github 连击要断了于是急中生智水的这一篇博客中要介绍的东西：

http://shields.io

打开页面就能发现，各种例子相当全面，完全不用我介绍

于是这篇文章就结束了

当然不会就这么结束！

虽然页面上各种第三方服务的例子很多，不过页面底部还是有自定义样式的说明的

比如颜色啦文字啦什么的

稍微折腾一下，就能弄出这种效果:

具体是这样写的:

https://img.shields.io/github/followers/iovxw.svg?style=social &label=Follow%20@iovxw&maxAge=2592000&logo=image%2Fpng%3Bbase64%2...%3D

  • 显示 iovxw 的 github followers
  • social 样式
  • 将左侧标签替换为 Follow @iovxw
  • 缓存时间 2592000 秒
    • 也就是刷新时间
  • base64 编码后的左侧 logo
    • 找一张高度大于等于 14 像素的图片就行，不过太大的话链接会非常非常长
    • 编码为 base64 可以用 这个
    • 然后还不能直接加上，先编码为 URL 才行
    • 同样可以用 在线工具 完成
    • 头部的 data: 其实可以省略，省 5 个字也是省嘛
    • 还可以加 link 字段来指定点击左侧和右侧时候的链接，不过嵌入在 HTML 里的时候没用，就不加了

觉得很厉害是不是？（才怪）

其实 github 早就有 这个 了（哈哈哈）
